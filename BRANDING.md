# MinaFox Brand Use

This project is open source, but the MinaFox name and brand identity need to stay clear so users can tell official MinaFox builds from forks, experiments, and downstream packages.

MinaFox, the MinaFox logo, and related mascot artwork are project brand assets.

## Code license vs. brand permission

The source code license grants rights to use, copy, modify, and redistribute covered code under the terms in `LICENSE`.

That code license does not grant permission to use MinaFox brand assets, the MinaFox logo, or the MinaFox mascot to identify modified, unofficial, or redistributed builds as official MinaFox builds.

## Allowed uses

You may use the MinaFox name when the use is truthful and not confusing, for example:

- referring to the upstream project as MinaFox;
- saying that a patch, package, fork, guide, or issue is "for MinaFox";
- linking to the official MinaFox repository;
- describing compatibility, such as "works with MinaFox";
- using screenshots of unmodified official MinaFox builds in reviews, tutorials, or bug reports.

## Uses that need permission

Please ask before using MinaFox branding in ways that could make users think your work is official. Without permission, do not:

- distribute modified builds under the MinaFox name or logo;
- use the MinaFox logo or mascot as the primary branding for a fork;
- register confusingly similar domains, packages, app-store listings, or social handles for browser/software projects;
- imply endorsement, certification, sponsorship, or official status by the MinaFox project;
- use MinaFox branding in a way that harms the project, misleads users, or hides privacy/security-impacting changes.

## Fork and package naming

Forks and downstream packages should use a clearly distinct name unless they are distributing an unmodified official MinaFox build or have permission.

Good examples:

- "AuroraFox, based on MinaFox"
- "MinaFox patch set for experimental sidebar work"
- "Unofficial test build based on MinaFox"

Avoid names that look official, such as:

- "MinaFox Pro"
- "MinaFox Browser Official"
- "MinaFox Nightly" unless it is an official release channel

## Mozilla and Firefox notice

Firefox is a trademark of Mozilla Foundation. MinaFox is independent and is not affiliated with or endorsed by Mozilla. It is also not sponsored by Mozilla.

MinaFox currently wraps the system Firefox binary with a separate profile, launcher, policy, icon, start page, and helper services. It does not ship a modified Firefox binary in the current standalone-wrapper phase.
