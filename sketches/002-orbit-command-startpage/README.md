## Variant: Orbit command startpage

### Design stance
A command-center new tab page: search first, then calm launch pads, reminders, and private/local services.

### Key choices
- Layout: browser chrome header plus startpage with central search orb, command buttons, and right-side glance panel.
- Typography: compact labels with a large friendly greeting.
- Color: cosmic gradient background with lavender cards and rose CTA.
- Interaction: chips filter quick links; privacy toggle changes status and glow.

### Trade-offs
- Strong at: startpage as the daily home base, easy search, and low-overwhelm next actions.
- Weak at: less opinionated browser chrome; mostly focuses on new tab design.

### Best for
- The immediate profile-layer MinaFox startpage where SearXNG/search and quick links matter most.
