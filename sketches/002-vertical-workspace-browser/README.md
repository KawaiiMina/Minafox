## Variant: Vertical workspace browser

### Design stance
Arc-like side rail for workspace switching, with a soft MinaFox glass startpage as the content canvas.

### Key choices
- Layout: full browser mockup with left workspace rail, tab stack, address pill, and page viewport.
- Typography: rounded system UI with oversized calm startpage title.
- Color: dark plum, lavender glass, pink active states.
- Interaction: workspace buttons switch dashboard copy; tab rows and quick cards hover.

### Trade-offs
- Strong at: making MinaFox feel like its own browser rather than only a themed Firefox profile.
- Weak at: real vertical workspace behavior needs Sidebery/extension/source work later.

### Best for
- A future MinaFox direction centered on workspaces, ADHD-friendly grouping, and low visual noise.
