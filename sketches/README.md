# MinaFox mockup sketches

Additional disposable HTML mockups generated for comparing browser-shell and startpage directions.

## Round 002

- `002-vertical-workspace-browser/index.html` — Arc-like vertical workspace browser shell plus MinaFox startpage canvas.
- `002-orbit-command-startpage/index.html` — startpage-first command center with search orbit and glance panel.
- `002-kawaii-compact-browser/index.html` — denser playful browser chrome with side widgets and mood modes.

Open on Linux:

```bash
xdg-open sketches/002-vertical-workspace-browser/index.html
xdg-open sketches/002-orbit-command-startpage/index.html
xdg-open sketches/002-kawaii-compact-browser/index.html
```

These are sketches, not production MinaFox code.
